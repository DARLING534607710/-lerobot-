
# 基于电位器的机器人关节数据采集与遥操作系统

## 📖 项目简介

本项目提供了一套低成本、高精度的机器人主从控制解决方案。通过**电位器（Potentiometer）** 作为核心传感器，构建了一个轻量级的主手关节数据采集装置，用于实时捕获人体关节的运动角度，并实现对机械臂的精准**遥操作（Teleoperation）**。

相比于传统昂贵的编码器或动作捕捉系统，本项目利用模拟电位器极大地降低了硬件成本，特别适用于机器人示教编程、远程操控研究及Lerobot数据采集

## ✨ 主要特性

- **低成本硬件方案**：核心传感器采用高精度旋转/直线电位器，成本低，易于获取。
- **实时数据采集**：基于windows平台的高频ADC采样，实时读取关节角度变化。
- **低延迟遥操作**：主从端通过[CAN]通信，实现从手对主手的实时随动。
- **直观的标定流程**：内置电位器校准与零点标定功能，适应不同机械结构。
- **模块化设计**：主手数据采集模块与从手驱动模块解耦，方便接入不同类型的机器人。

## 🛠️ 硬件架构

- **主控芯片**：[例如：STM32F103C8T6 / Arduino Mega / ESP32]
- **传感器**：旋转电位器（或直线电位器），用于捕捉关节角度。
- **机械本体**：3D打印/轻量化铝制外骨骼结构。
- **通信接口**：[CAN收发器]
- **从端机器人**：[例如：Dobot / UR5 / 自研舵机机械臂]

## 📡 系统工作流

1. **主手佩戴**：操作者将电位器关节用棍状物连接
2. **信号转换**：关节转动带动电位器阻值变化，主控通过ADC读取电压并换算为角度值。
3. **数据传输**：角度数据打包后通过通信模块发送至上位机或直接从站。
4. **运动映射**：从端机器人解析数据包，驱动舵机/电机复现主端关节运动。
